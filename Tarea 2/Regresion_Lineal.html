Regresión Lineal¶

1. Introducción¶

La Regresin Lineal es una de las herramientas fundamentales del anlisis estadstico y economtrico que se utiliza en la actualidad. Este modelo permite relacionar de manera lineal una variable dependiente (Y) con un conjunto de variables explicativas (X), con el objetivo de predecir el valor promedio de la variable dependiente Y a partir de valores conocidos de X (Gujarati, D., & Porter, D., 1999).

A lo largo de este documento se presentar formalmente el modelo de regresin lineal, y se ilustrar su aplicacin con un conjunto de datos real.

2. Historia de la Regresión Lineal¶

Como se mencion anteriormente, la Regresin Lineal es una de las herramientas fundamentales del anlisis estadstico que se utiliza en la actualidad. Su capacidad para extraer conocimiento a partir de datos y utilizarla para hacer predicciones de datos desconocidos ha permitido que sea utilizada en diferentes reas del conocimiento, como en economa, psicologa, bioestadstica, etc. Sin embargo, sus orgenes son bastante interesantes, pues se remontan al S. XIX y provienen del estudio de la astronoma, la geodesia y la gentica.

Segn Gujarati, D., & Porter, D., (1999), Galton public el primer artculo que mencionaba el trmino regresin a la media. Segn Galton, existe una tendencia gentica para que padres altos engendraran hijos altos, mientras que los hijos de padres bajos fueran de una baja estatura. Sin embargo, la estatura promedio de los hijos de padres de la misma estatura tendran a "regresar" a la media de la poblacin.

Curiosamente, el mtodo con el cual se podra analizar relaciones similares a las de Galton, fue desarrollado en 1805 por Adrien-Marie Legendre y posteriormente en 1809 por Carl Friedrich Gauss[1]. El objetivo de Legendre fue combinar la informacin de osbservaciones para determinar las rbultas de cometas. En su libro Nouvelles mthodes pour la dtermination des orbites des comtes, publicado en 1805, Legendre desarrolla el mtodo de mnimos cuadrados para ajustar un ecuaciones lineales utilizando toda la informacin disponible.

Un ao despus, Gauss publicara un artculo en el que utilizaba "su" mtodo de Mnimos Cuadrados para calcular las rbitas de cuerpos Celestes. Aunque Gauss tuvo una disputa con Legendre por quin haba desarrollado anteriormente el mtodo, la publicacin de Gauss fue un paso ms all al relacionar la regresin con la distribucin normal, gracias a lo que se conoce como las ecuaciones normales.

Hoy en da se siguen utilizando los mtodos desarrollados por Legendre y Gauss para realizar predicciones en diferentes reas del conocimiento, descubrir informacin de cmo se relacionan entre s un conjunto de variables, e incluso construir nuevos mtodos de prediccin. Si bien los aportes de Gauss tuvieron un componente fuerte en trminos de probabilidad, en este notebook nos basarmos principalmente en la estimacin del modelo, obviando sus propiedades estadsticas.

3. Planteamiento del Modelo¶

Hasta este momento se ha mencionado que el modelo de regresin lineal permite encontrar la relacin lineal entre una variable dependiente ($Y$) y una variable o un conjunto de variables explicativas ($X$). En esta seccin se discutir la forma funcional del modelo, su estimacin y su utilizacin como modelo de prediccin.

Para comenzar, asuma que un modelador tiene una variable de inters $y_i$ que es medida para N individuos de una poblacin, de forma que $i = 1, 2, \dots, N$. Adicional a este, el modelador cuenta con informacin adicional de los individuos, $x_i$, que est relacionada con la variable de inters. Cuando el modelador grafica sus datos utilizando un diagrama de dispersin, obtiene lo siguiente:

El modelador se da cuenta que hay una tendencia: valores ms grandes de la variable $X$ estn relacionados con valores ms grandes de la variable $Y$. Adicionalmente, se da cuenta que este patrn parece ser constante; es decir, por un mismo aumento en $X$, la variable $Y$ aumenta la misma cantidad (en promedio). Teniendo en cuenta lo anterior, el modelador considera que se puede ajustar una lnea recta que atraviese estos puntos. La ecuacin que describira este tipo de modelos vendra dada por la ecuacin de la recta,

$$y_i = \beta_0 + \beta_1 x_i + \varepsilon_i$$

donde $\beta_0$ es el intercepto de la recta y $\beta_1$ es su pendiente. Sin embargo, existen muchos coeficientes $\beta_0$ y $\beta_1$ que permitan trazar una recta que pase por estos puntos. A continuacin se muestran dos ejemplos de posibles rectas.

Se puede observar que ambas rectas presentan un comportamiento similar, y se puede argumentar que ambas ecuaciones (o ambos modelos) recogen adecuadamente el comportamiento de los datos. Por lo anterior, es necesario definir un criterio bajo el cual determinar el modelo ms adecuado.

En principio, uno de los criterios ms utilizados es que la ecuacin de la recta de regresin sea aquella que minimice una medicin de total de error. En particular, si se define la suma de errores de un modelo como,

$$S = \sum_{i = 1}^{N} (y_i - \beta_0 - \beta_1 x_i)^2 = \sum_{i = 1}^{N} e_i^2$$

entonces se debe encontrar las cantidades $\beta_0$ y $\beta_1$ que la minimice.

Si este problema se reescribe matricialmente como,

$$Y = \begin{bmatrix} y_1\\ y_2 \\ \vdots \\ y_n \end{bmatrix} \qquad X = \begin{bmatrix} 1& x_1\\ 1 & x_2 \\ \vdots & \vdots \\ 1 & x_n \end{bmatrix} \qquad \beta = \begin{bmatrix} \beta_0\\ \beta_1 \end{bmatrix} \qquad e = \begin{bmatrix} \varepsilon_1\\ \varepsilon_2 \\ \vdots \\ \varepsilon_n \end{bmatrix}$$

entonces el modelo podra reescribirse como, $$Y = X\beta + e$$

y la cantidad a minimizar es,

$$S = (Y - X\beta)'(Y - X\beta) = e'e$$

Para encontrar $\beta$ se utiliza el criterio de Mnimos Cuadrados Ordinarios. De esta forma, al derivar $S$ respecto a los parmetros e igualando a cero, es decir,

$$\frac{\partial S}{\partial \beta} = 0$$

Se obtiene que el valor de $\beta$ que minimiza $S$ corresponde a,

$$\hat{\beta} = (X'X)^{-1} X'Y$$

A partir de all, se obtiene que el modelo de regresin lineal para estos datos corresponde a,

$$\hat{y}_i = \hat{\beta}_1 + \hat{\beta}_2 x_i $$

Lo anterior puede generalizarse para cuando se tengan ms variables explicativas.

A continuacin se construir una clase que permite estimar un modelo de regresin y utilizarlo para hacer prediccin sobre un conjunto de datos.

import numpy as np

class LinearRegression:

    def __init__(self, covariates, target):
        self.covariates = covariates
        self.target = target

        self.design_matrix = self.__build_design_matrix(self.covariates)
        self.coefficients = self.fit()


    def __build_design_matrix(self, raw_input):
        """
        Construye la matriz de diseo aadiendo una columna de unos. 

        Params
        --------
          raw_input (np.ndarray):
            Un arreglo de numpy de dimensin NxK que se le desea agregar una columna 
            de unos.

        Returns
        --------
          design_matrix (np.ndarray):
            Un arreglo de numpy de dimensin Nx(K+1) que contiene una columna de unos 
            en la primera posicin.
        """
        data_shape = raw_input.shape
        ones = np.ones(data_shape[0]).reshape(-1, 1)
        design_matrix = np.append(ones, raw_input, axis = 1)
        return design_matrix

    def fit(self):
        """
        Realiza la estimacin de los coeficientes de una regresin lineal.
        Bhat = (X'X)^{-1} X'Y
        """
        XTX = np.dot(self.design_matrix.T, self.design_matrix)
        inv_XTX = np.linalg.inv(XTX)
        XTY = np.dot(self.design_matrix.T, self.target)
        b_hat = np.dot(inv_XTX, XTY)
        return b_hat 


    def predict(self, data):
        """
        Realiza la prediccin sobre un conjunto de datos.

        Params
        -------
          data (np.ndarray):
            Un arreglo de numpy que contiene informacin de las covariables del modelo

        Returns
        -------
          yhat (np.ndarray):
            Un arreglo de numpy que contiene las predicciones
        """
        X = self.__build_design_matrix(data)
        y_hat = np.dot(X, self.coefficients)
        return y_hat

def MSE(y_true, y_pred):
    N = y_true.shape[0]
    diff = (y_true - y_pred)**2
    ssr = diff.sum()
    return ssr/N
